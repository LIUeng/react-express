# react-express
#从零开始使用webpack搭建起一个react+express的框架
  功能：打造一个前后端分离的脚手架
  express 是Node的一个web框架 连接数据库操作 后端
  react 前端框架
#使用
  第一步：安装依赖npm install
  第二步：npm start
  第三步：打包npm run build
  
#侧重点
  热更新 & 热加载
  使用webpack-dev-middleware 配合 webpack-hot-middleware 插件使用
  详情可以仔细阅读webpack4.0文档
